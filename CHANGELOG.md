# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.2] - 2026-02-15
### Fixed
- Templating system and CLI scaffolding bugs.

## [0.1.1] - 2025-12-20
### Added
- Windows support via `pythonnet` and WebView2.
- Cross-platform architecture with `PaneerBase`.
- `emit` functionality for backend-to-frontend events.
- `paneer build` improvements (onefile, noconsole).

### Fixed
- Package discovery issues in `pyproject.toml`.

## [0.1.0] - 2025-09-12
### Added
- Initial release.
- Basic Linux support using GTK4 and WebKit.
- CLI for scaffolding projects.
